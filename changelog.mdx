## Migration Guide

Follow these steps to upgrade to the latest version.

<Steps>
  <Step title="Backup Data" icon="database">

    Create a full backup of your Brooma instance.

````bash
brooma backup --path=/backups/latest
````

  </Step>
  <Step title="Update Package" icon="download">

    <Tabs>
      <Tab title="npm" icon="package">
````bash
npm install brooma@latest
````
      </Tab>
      <Tab title="Docker" icon="docker">
````bash
docker pull brooma/brooma:latest
docker-compose up -d
````
      </Tab>
    </Tabs>

  </Step>
  <Step title="Run Migrations" icon="settings">

    Apply database migrations.

````bash
brooma migrate --env=production
````

  </Step>
  <Step title="Verify" icon="check-circle">

    Test key features like task assignment and notifications.

  </Step>
</Steps>

<Callout kind="tip">
  For v1.2.0, update your webhook URLs to use `https://api.example.com/v1/webhooks`.
</Callout>
