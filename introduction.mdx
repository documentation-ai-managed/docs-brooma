## Overview

Brooma provides a centralized space for you to organize, manage, and collaborate on all your project documentation. Create structured docs for teams, track changes, and ensure everyone stays aligned. Whether you handle service requests, legal workflows, or customer support, Brooma simplifies documentation across your projects.

You gain full control over content with version history, search, and sharing options. Integrate with your tools to keep documentation live and up-to-date.

## Quick Start

Get up and running in minutes with these steps.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://dashboard.brooma.com/register` with your email.
  </Step>
  <Step title="New Space" icon="plus">
    Click "New Documentation Space" and name it for your project.
  </Step>
  <Step title="Add Content" icon="edit-3">
    Use the editor to create pages. Add markdown, images, or embeds.
  </Step>
  <Step title="Share" icon="share-2">
    Generate a shareable link or invite team members.
  </Step>
</Steps>

<CodeGroup tabs="Markdown,Embed">
  ```markdown
  # Project Overview

  Welcome to your Brooma space.

  - [ ] Task 1
  - [x] Task 2
  ```
  ```html
  <iframe
    src="https://docs.brooma.com/your-space"
    width="100%"
    height="600"
  />
  ```
</CodeGroup>
